#Cygni competence - 7 databases

This course is based on the book [Seven Databases in Seven Weeks](https://pragprog.com/book/rwdata/seven-databases-in-seven-weeks).

Riak is removed in favour of ElasticSearch.

| Date       | Database        | Trainer               |
| ---------- | --------------- | --------------------- |
| 2016-11-17 | CouchDB         | Daniel Lenksjö        |
| 2016-12-12 | Neo4J           | Johannes Dolk         |
| 2017-01-26 | Redis           | Emil Bergström        |
| 2017-02-15 | PostgreSQL      | Rickard Lindström     |
| 2017-03-09 | MongoDB         | Fortunato Flores Ando |
| 2017-03-27 | HBase           | Pär Tjärnberg         |
| 2017-04-27 | ElasticSearch   | Henrik L. + Breding   |

Each database has its own folder. Example code and data files are located
in a subfolder called `code`. The `README.md` file for each database
provides more information.
